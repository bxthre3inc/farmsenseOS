# Knowledge Item: LRZ1 (Truth Node / Grounding Raster) V1.21

## Role: High-Fidelity Monolithic Ground Truth Node

**Deployment Density**: **12 per field** (2:4:12 Stereo Ratio).
The LRZ1 is the monolithic calibration reference. It provides the ground-truth moisture values that the RSS uses to validate and correct the virtual sensor grid generated by Kriging.

## Physical Stack: 18U Alpha-Sled (Monolithic Configuration)

| Slot | Depth | Sensor |
|:---|:---|:---|
| 1 | Top | 1U Stamped Desiccant Matrix |
| 2–5 | — | 4U Primary Battery (21700 Hybrid Cell) |
| 10 | 10" | 1U Sensor (10" Depth) |
| 18 | 18" | 1U Sensor (18" Depth) |

## Sensing Physics: "Fringe Field" Capacitance

- **Excitation Frequency**: 80MHz oscillator.
- **Dielectric Barrier**: HDPE SDR9 Shell + 2mm Nitrogen gap.
- **Calibration Matrix**: 5-point factory curve mapped against **SLV Soil Series 101-B**.

## Electronics

- **MCU**: ASR6601 LoRa SoC (Cortex-M4 + integrated SX1262).
- **Encryption**: AES-128/256 hardware crypto.
- **State Machine**:
  - Deep Sleep: 1.5µA (99.9% of duty cycle)
  - Sample Mode: 3.5mA (40ms dielectric ping)
  - TX Chirp: 120mA (10ms burst)

## GPIO Pinout (AlphaSled PCBA)

| Pin | Function | Peripheral |
|:---|:---|:---|
| P0.02 | AIN0 | Dielectric Sensor 1 (10") |
| P0.03 | AIN1 | Dielectric Sensor 2 (18") |
| P0.28–P0.31 | SPI | LoRa SX1262 |
| P1.01 | BMS_INT | Battery Management Interrupt |
