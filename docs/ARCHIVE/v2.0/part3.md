Methodology\n\n### 4.4.1 Edge-EBK (Empirical Bayesian Kriging)\n\n**Level 1.5 (PMT):** 50m resolution, IDW fallback\n- Hardware: ESP32-S3, FPU accelerated\n- Algorithm: Modified IDW with anisotropic distance\n- Latency: <50ms for 16x16 matrix\n\n**Level 2 (DHU):** 20m/10m resolution, Ordinary Kriging\n- Hardware: Jetson Orin Nano (CUDA)\n- Algorithm: Ordinary Kriging with spherical variogram\n- Parameters:\n  - Nugget: 0.0012\n  - Sill: 0.0085\n  - Range: 245m\n\n**Level 3 (RSS):** 1m resolution, Regression Kriging\n- Hardware: Threadripper PRO (64 cores)\n- Algorithm: Regression Kriging with Sentinel-2 NDVI covariates\n- Covariates: Satellite NDVI, thermal, elevation, slope, aspect\n- Cross-validation: R² = 0.94, MAPE <5%\n\n### 4.4.2 Matérn Kernel Specification\n\n**Why Matérn over exponential/spherical:**\n- Allows non-integer smoothness parameter ν\n- SLV soil moisture exhibits localized discontinuities (compaction ridges, wheel tracks)\n- ν auto-tunes 0.5 (rough/exponential) to 2.5 (smooth) based on Field Roughness Index (FRI)\n\n**Formula:**\n```\nZ(s) = m(s) + ε(s)\n```\nWhere m(s) = deterministic trend (satellite covariates), ε(s) = spatially correlated residual\n\n---\n\n## 4.5 Adaptive Recalculation Engine: "Fisherman's Attention"\n\n| Mode | Trigger | Frequency | Power Draw |\n|------|---------|-----------|------------|\n| **SLEEP** | Stable
